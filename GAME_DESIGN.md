# 踹他一百下 - 游戏设计文档

## 一、游戏概述

**游戏名称**：踹他一百下

**游戏类型**：3D潜行/动作游戏

**技术栈**：Vue 3 + TypeScript + Babylon.js 9 + Vite

**运行方式**：本地开发服务器 (端口 3000)，局域网可访问

---

## 二、游戏背景

有一个神人，每天提供花里胡哨的建议折磨你。你需要在他不注意时偷偷踹他，踹满一定次数即可过关。但要小心，如果被他发现，你会受到惩罚！

---

## 三、核心玩法

### 3.1 游戏目标

- 通过关卡制推进游戏
- 每关需要踹够指定次数才能过关
- 被发现3次（生命值归零）则游戏失败

### 3.2 关卡设计

| 关卡  | 需要踹的次数 |
| ----- | ------------ |
| 第1关 | 10脚         |
| 第2关 | 20脚         |
| 第3关 | 35脚         |
| 第4关 | 50脚         |
| 第5关 | 70脚         |
| 第6关 | 100脚        |
| 第7关 | 150脚        |

## 四、操作说明

### 4.1 移动控制

| 按键 | 动作     |
| ---- | -------- |
| W    | 向上移动 |
| S    | 向下移动 |
| A    | 向左移动 |
| D    | 向右移动 |

### 4.2 其他操作

| 按键       | 动作                         |
| ---------- | ---------------------------- |
| 鼠标左键   | 踹击/武器挥砍（靠近神人时） |
| 鼠标右键   | 蓄力投掷武器（按住蓄力，松开发射） |
| 空格       | 举起键盘挡脸（持续5秒，50%减伤） |
| 1/2/3      | 使用道具栏中的道具           |

### 4.3 视角

- 固定俯视角（类等距视角）
- 相机自动跟随玩家移动
- 无需鼠标控制视角

## 五、神人AI系统

### 5.1 神人状态

```
┌──────────┐    随机触发    ┌──────────┐
│  正常状态  │ ──────────→ │ 手机闪烁  │
│ (用电脑)  │              │ (预警期)  │
└──────────┘              └──────────┘
      ↑                          │
      │                    3-5秒后
      │                          ↓
      │                    ┌──────────┐
      └─────────────────── │  回头状态  │
           动画完成后       │ (检查)    │
                          └──────────┘
```

### 5.2 回头动画流程

| 阶段 | 动作     | 时间  | 说明                   |
| ---- | -------- | ----- | ---------------------- |
| 1    | 站起     | 0.5秒 | 身体从椅子上升起       |
| 2    | 转身     | 0.4秒 | 身体旋转180度面向玩家  |
| 3    | 摇头观察 | 2秒   | 头部左右摇晃，检测玩家 |
| 4    | 转回     | 0.4秒 | 身体旋转回原位         |
| 5    | 坐下     | 0.5秒 | 身体下降回到椅子       |

### 5.3 预警机制

- 手机闪烁红色光效
- 屏幕边缘红色闪烁警告
- 状态栏显示"手机闪烁"提示
- 预警时间：3-5秒（随机）

### 5.4 检测规则

神人在"摇头观察"阶段会检测玩家：

- 玩家在躲藏点后方 → 安全
- 玩家使用键盘挡脸 → 50%概率安全
- 玩家使用隐身药水 → 安全
- 其他情况 → 被发现，扣1血

### 5.5 开会技能

- 每12-20秒随机触发，持续6秒
- 开会期间神人举起手臂挥手
- 所有方向的攻击均被弹开（50%概率消耗武器）
- 屏幕显示朱砂色警告边框

### 5.6 巡查技能

- 每20-35秒随机触发（60%概率）
- 5个阶段：预警(2.5s) → 站起(0.6s) → 巡查(8-12s) → 返回 → 坐下(0.6s)
- 巡查期间拥有视野锥检测（25°半角，5单位半径）
- 正面攻击无效（武器被弹开），背后偷袭有效
- 投掷物无论方向均被弹开
- 被视野扫到扣1血（1秒伤害冷却）

## 六、玩家系统

### 6.1 生命值

- 初始生命值：3条命
- 被发现一次扣1条命
- 生命值归零则游戏失败

### 6.2 键盘挡脸机制

- 按空格键举起键盘挡脸
- 持续时间：5秒
- 冷却时间：5秒（用完后）
- 效果：50%伤害减免
- 头部上方显示倒计时进度条

### 6.3 隐身药水

- 持续时间：5秒
- 剩余1.5秒时开始闪烁提醒
- 期间不被任何检测方式发现

## 七、道具系统

### 7.1 道具列表

| 道具     | 效果           | 持续时间 |
| -------- | -------------- | -------- |
| 加速鞋   | 移动速度x2     | 5秒      |
| 隐身药水 | 不被发现       | 5秒      |
| 噪音器   | 吸引神人注意力 | 3秒      |
| 力量手套 | 踹击计数+2     | 5秒      |

### 7.2 道具机制

- 每10秒随机刷新1个道具
- 道具出现在场景边缘
- 玩家靠近自动拾取
- 最多携带3个道具
- 按 1/2/3 使用对应道具

### 7.3 武器系统

| 武器     | 伤害 | 眩晕时间 | 挥砍范围 | 可投掷 |
| -------- | ---- | -------- | -------- | ------ |
| 狼牙棒   | 5    | 3秒      | 2.5      | ✓      |
| 棒球棒   | 3    | 2秒      | 2.8      | ✓      |
| 平底锅   | 1    | 1.5秒    | 2.0      | ✗      |
| 戒尺     | 1    | 1.0秒    | 2.2      | ✗      |

- 右键按住蓄力，松开发射投掷物
- 蓄力时显示力量条（正弦波振荡）
- 抛射物受重力影响，呈抛物线飞行

## 九、彩蛋模式（Easter Egg）

每关完成进入结算过渡画面时，可点击"奖励神人"按钮进入30秒限时FPS射击模式：

- **视角**：第一人称，只看到右手+武器
- **武器**：枪（连射）、火箭炮（抛射物+爆炸）、手榴弹（抛物线+延迟爆炸）
- **Boss**：在房间内左右走动，被击中有不同反应
  - 枪击中：原地抖动+停止移动+随机吐槽
  - 火箭炮/手榴弹击中：眩晕3秒
- **倒计时**：30秒结束后回到过渡画面

## 十、视觉设计

### 10.1 美学方向：昭和 Office 怪谈

1980年代东京某栋昏黄办公楼里，空调嗡嗡作响的深夜办公室。压抑、荒诞、黑色幽默。

### 10.2 设计规范

- **配色**：暖褐+苔绿+朱砂（无紫色、无Material Design默认色）
- **字体**：DotGothic16（标题）、Zen Kaku Gothic New（正文）、VT323（计数器）、Reggae One（印章）
- **阴影**：硬阴影（无模糊），仿丝网印刷质感
- **设计Token**：`src/styles/theme.css` 中定义所有CSS变量

---

## 十一、技术架构

### 11.1 引擎

- **Babylon.js 9**：`@babylonjs/core` + `@babylonjs/loaders`
- **物理**：手写碰撞检测 + 抛射物弹道
- **状态机**：通用 `StateMachine` 类，Enemy 6种状态
- **资源管理**：`AssetManager` GLB加载和缓存

### 11.2 HUD

- Vue 3组件覆盖在Babylon画布之上
- 5个子组件：TopBar、InventoryBar、StatusTicker、ThreatVignette、OverlayScreens
- `pointer-events: none` 穿透鼠标事件到3D画布
