# 踹他一百下

3D潜行/动作游戏 — 趁神人不注意，悄悄靠近踹他！

## 简介

某栋昏黄办公楼的深夜，那位"神人"又在那里指点江山。趁他低头，悄悄靠近——踹。

- 7个关卡，踹击目标从10到150递增
- 躲避神人的回头检查、开会、巡查等多种技能
- 收集道具和武器，正面挥砍或蓄力投掷
- 被发现3次则游戏失败
- 每关结算可进入彩蛋模式：30秒FPS射击小游戏

## 操作

| 按键 | 动作 |
|------|------|
| W/A/S/D | 移动 |
| 左键 | 踹击/武器挥砍 |
| 右键 | 蓄力投掷武器（按住蓄力，松开发射） |
| 空格 | 举起键盘挡脸（5秒，50%减伤） |
| 1/2/3 | 使用道具 |

## 技术栈

- **前端**: Vue 3 + TypeScript
- **3D引擎**: Babylon.js 9
- **构建**: Vite 5.4
- **包管理**: pnpm
- **部署**: GitHub Pages

## 开发

```bash
pnpm install    # 安装依赖
pnpm dev        # 启动开发服务器 (端口 3000)
pnpm build      # 类型检查 + 生产构建
pnpm preview    # 预览生产构建
```

## 项目结构

```
src/
├── babylon/              # 游戏引擎
│   ├── core/             # 引擎基础设施
│   ├── entities/         # 玩家和敌人
│   ├── state/            # 敌人状态机
│   ├── systems/          # 游戏子系统
│   ├── environment/      # 场景和躲藏点
│   ├── weapons/          # 武器模型
│   ├── easter-egg/       # 彩蛋模式
│   └── GameLoop.ts       # 主循环
├── components/           # Vue UI组件
│   └── hud/              # HUD子组件
├── styles/               # 设计Token
└── types/                # TypeScript类型定义
```
