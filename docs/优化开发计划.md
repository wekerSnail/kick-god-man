## 二、命中打击粒子（预计 1.5h）

**改动文件**：`src/game/GameLoop.ts`

**内容**：
1. 复用现有 `createVictoryParticles()` 的 `THREE.Points` 模式
2. 新增 `createHitParticles(position)` 方法，命中时喷射 10-15 个小粒子
3. 粒子颜色根据武器类型变化（狼牙棒=红、棒球棒=白、平底锅=灰）
4. 在攻击命中分支（`GameLoop.ts:168` 附近）调用

```ts
private createHitParticles(position: THREE.Vector3, color: number = 0xffd700) {
  const scene = this.sceneManager.getScene()
  const count = 12
  const geometry = new THREE.BufferGeometry()
  const positions = new Float32Array(count * 3)
  const velocities: number[] = []

  for (let i = 0; i < count; i++) {
    positions[i * 3] = position.x
    positions[i * 3 + 1] = position.y + 1
    positions[i * 3 + 2] = position.z
    velocities.push(
      (Math.random() - 0.5) * 6,
      Math.random() * 5 + 2,
      (Math.random() - 0.5) * 6
    )
  }

  geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3))
  const material = new THREE.PointsMaterial({
    size: 0.1,
    color,
    transparent: true,
    opacity: 1
  })
  const particles = new THREE.Points(geometry, material)
  particles.userData.velocities = velocities
  particles.userData.life = 0
  scene.add(particles)
  this.hitParticles.push(particles)
}
```

**验收**：踢中/挥砍命中时有粒子喷射效果，0.4 秒后消失。

## 三、GLTF 模型替换 — 第一批：静态道具（预计 2h）

**改动文件**：`src/game/Props.ts`、`src/game/HidingSpots.ts`

**前置**：从 Kenney / Poly Pizza 下载低模 `.glb` 文件，放入 `public/models/`

**内容**：
1. 在 `ResourceCache.ts` 新增 GLTF 加载能力（不替换现有，扩展现有）
2. `Props.ts` 的道具生成改为优先用 GLTF 模型，fallback 到程序化几何
3. `HidingSpots.ts` 的盆栽/柜子/沙发替换为 GLTF 模型

```ts
// ResourceCache.ts 新增
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader.js'

private static gltfLoader = new GLTFLoader()
private static modelCache = new Map<string, THREE.Group>()

static async preloadModels(): Promise<void> {
  const models: Record<string, string> = {
    plant: '/models/plant.glb',
    cabinet: '/models/cabinet.glb',
    sofa: '/models/sofa.glb',
  }
  for (const [key, url] of Object.entries(models)) {
    const gltf = await this.gltfLoader.loadAsync(url)
    gltf.scene.traverse(child => {
      if ((child as THREE.Mesh).isMesh) {
        child.castShadow = true
        child.receiveShadow = true
      }
    })
    this.modelCache.set(key, gltf.scene)
  }
}

static getModel(key: string): THREE.Group | null {
  const model = this.modelCache.get(key)
  return model ? model.clone() : null
}
```

**验收**：盆栽/柜子/沙发显示为低模 3D 模型，阴影正常。
